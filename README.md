# ASP MVC Structure using .NET Core

## Steps Taken
1. Create MvcMovie app directory
```
mkdir MvcMovie
cd MvcMovie
dotnet new mvc
```

2. Press `Debug (F5)` to build and run the program.

3. App is started at `localhost:5000`
